## 软件介绍  

+  `MikuBeat` 是一款安卓 **术力口** 播放器软件，基于 `AndroLua` - `OpenLuaX+` 开发，完全 `免费开源` 。  
+ MikuBeat依赖于网易云音乐、哔站，通过开放接口与网页爬虫获取音频与歌曲信息，项目仅用于交流学习，无意对相关图片与歌曲造成任何侵权。  
+ MikuBeat坚持使用 `Material Design 3` 主题开发，界面优雅美观。
